# i-ching-5d-worldline-engine

**用现代计算重新启动三千年前的宇宙认知操作系统。**

我们在此提出一个探索性的理论假说：《易经》是一套被遗忘的"宇宙认知操作系统"。其大衍筮法的推演过程，与 Filip Strubbe（2025）提出的**结晶化时空**物理学框架在结构上惊人一致——后者认为量子现象并非自然界的本质，而是四维时空在更高层次上演化的投影。

**本项目以实验为先导。** 我们先完成了 V3 至 V27 的完整实验序列，在真实世界时序预测任务上验证了系统有效性，随后独立交叉比对了上述物理学框架，确认了两者在算法结构上的同构性。实验结果表明：**每当我们还原《易经》的一个完整结构元素，系统在预测任务上的性能都获得了可测量的提升。**

我们不宣称构建了一个更好的大语言模型，而是指出了一个被长期忽视的方向：三千年前的东方智慧，可能已经编码了宇宙认知的底层算法。

---

## 内容边界说明

为避免概念混淆，本仓库所有内容按以下三层划分：

1. **已验证实验结论（客观、可复现）**
   基于《易经》卦变、互卦、卦气等结构搭建的推理系统，组件完整性与预测性能呈显著正相关。V3–V27 对照实验、消融实验已完成量化验证。

2. **结构同构推论（可数学验算）**
   本项目的推演架构与 Strubbe 的结晶化时空框架，在状态空间划分、演化算子、高维投影、多观测者共识逻辑上存在系统性结构同构。该结论由实验完成后独立交叉比对得出，非前置概念包装。

3. **探索性理论假说（开放性研究方向）**
   "《易经》为宇宙认知操作系统"是本项目提出的延伸假说，**目前尚未经过物理实验证实**，仅作为跨学科探索方向。

## 项目架构

所有模块与论文中的数学定义一一对应：

- **卦象生成**：熵池随机源 + 大衍筮法模拟 → 卦象编码
- **三陈推演**：体用（五行生克）、时位（动爻位置）、卦变（互卦变卦）——三维独立推演后内部共识
- **五维世界线追踪**：多条世界线概率分布，贝叶斯在线更新
- **多观测者共识**：五位独立卦师，先验分化，多数投票
- **贝叶斯在线学习**：Dirichlet 后验更新 + EMA 准确率追踪
